## 🚀 Featured Project

### 🐝 SSH Honeypot Dashboard

A fake SSH honeypot built for security research and lab testing.
It captures connection attempts, logs attacker behavior, stores events, and displays activity through a simple dashboard.

**Main Ideas:**

* Simulate an exposed SSH service
* Capture login attempts and attacker IPs
* Store attack logs in a database
* Display activity in a dashboard
* Practice defensive visibility from an attacker-focused perspective

**Tech Stack:**

Python, Flask, SQLite, Linux, Networking, MITRE ATT&CK Concepts
